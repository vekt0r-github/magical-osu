# Main

This file serves as a summary/directory of all information in the `/wiki` directory.

## Files

- `visuals.md`: Information about the site's appearance belongs here. This includes art, textures, layout, and related information. Anything gameplay related is not described in this file.

- `gameplay.md`: Key information about any gameplay exists in this file. Gameplay mechanics, concepts, and core details for the gameplay aspect of the website are specified. Additionally, gameplay visual information is detailed here.
